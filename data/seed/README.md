- `cl_law_19628.json` — Chile Law 19.628 provisions

- `mx_lfpdppp.json` — Mexico LFPDPPP provisions
- `cr_law_8968.json` — Costa Rica Law 8968 provisions

Each file should follow the SeedSource interface defined in `scripts/build-db.ts`.
