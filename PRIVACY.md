# Privacy Policy

## Data Processing

This MCP server is **stateless**. It does not:
- Log, store, or transmit user queries
- Track usage or collect analytics
- Process personal data beyond the legislative texts in its database

## Local vs. Remote

- **Local (stdio):** All processing happens on your machine. No network calls.
- **Remote (Streamable HTTP):** Queries are processed server-side but not logged or stored.

## Legislative Content

The database contains publicly available legislative texts from official government sources. No personal data is included in the dataset.
